# auction-neural-networks
Modelagem de redes neurais utilizando Perceptron Multicamadas (MLPRegressor), aplicados ao dataset "Auction Verification" para investigar a possibilidade de substituir a verificação custosa de modelos de processos complexos (como leilões simultâneos de múltiplas rodadas para a venda de espectros de frequência) por previsões dos resultados.

⚠️ Aviso: O arquivo .ipynb está passando por revisão devido a problemas técnicos. Recomendo a consulta à versão final em PDF, que contém todo o código e a análise completa do trabalho."
